## 栈

栈是一种线性数据结构，有以下两种特点：

- 按照“后进先出”的顺序访问；
- 只能对栈顶元素进行操作。

栈的三种常用操作：

1. push：向栈顶增加元素；
2. pop：返回栈顶元素并删除该元素；
3. peek：返回栈顶元素，不删除。
